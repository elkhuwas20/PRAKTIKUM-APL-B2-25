📞 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

If build.bat fails:
  1. Check g++ is installed: g++ --version
  2. Check file paths are correct
  3. Check file permissions
  4. Try manual compilation

If colors don't show:
  1. Run on Windows console (cmd.exe)
  2. Enable ANSI support: reg add HKCU\Console /v VirtualTerminalLevel /t REG_DWORD /d 1
  3. Update Windows

If Unicode characters appear wrong:
  1. Set console to UTF-8: chcp 65001
  2. Change font to one supporting Unicode
  3. Resize console window

Program crashes on login:
  1. Check username/password length (< 30 chars)
  2. Check jumlahUser is correctly initialized
  3. Enable debug output to trace issue
